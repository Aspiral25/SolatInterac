## About Solat Interactive

Solat Interactive is a desktop application (PyQt6 + Flask) designed to display Muslim prayer times on dedicated screens at mosques and surau. It fetches official prayer times from the Malaysian JAKIM e-Solat API, plays Adhan/Iqamah audio alerts, and provides a remote image manager via a built-in web server.

Licensed for **Surau Al-Huda IPD BaratDaya**.
