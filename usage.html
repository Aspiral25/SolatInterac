Usage Guide

Solat Interactive runs in fullscreen mode by default, displaying prayer times, a live countdown, Islamic events, and a rotating slideshow. This guide covers the main user experience and all available features.

Main Display

The display is divided into two primary zones:

  • Left panel: Rotating slideshow images with configurable transitions.
  • Right panel: Prayer times grid, current clock, date display (Gregorian + Hijri), live countdown timer, and upcoming Islamic events.

Prayer Time Grid

The right panel displays seven daily prayer periods:

PrayerDescription
FajrDawn prayer (Subuh)
SyurukSunrise time
DhuhaLate morning
DhuhrMidday prayer (Zohor)
AsrAfternoon prayer (Asar)
MaghribSunset prayer (Maghrib)
IshaNight prayer (Isyak)

Times are fetched from the Malaysian JAKIM e-Solat API and cached locally. The current and next prayer are highlighted automatically.

Countdown Timer & Pre-Adhan Alert

A live countdown shows the time remaining until the next prayer. Three minutes before prayer:

  • The countdown begins pulsing with a color animation.
  • A configurable pre-Adhan alert triggers at the specified lead time (default: 10 minutes).
  • At the exact prayer time, the full-screen Adhan sequence begins.

Iqamah Sequence

When a prayer time is reached, the app enters a three-stage sequence:

  1. Adhan Phase: A full-screen Adhan GIF displays for a configurable duration.
  2. Iqamah Countdown: After a delay, an animated overlay fades in showing an Iqamah countdown (MM:SS). The duration is set per-prayer.
  3. SOLAT Alert: When the countdown reaches zero, a glow-style "SOLAT" announcement screen appears and holds for a configurable final delay.

Audio (Adhan and Iqamah) plays automatically during these phases at the configured volume level.

Slideshow

The left panel continuously rotates through images placed in the slideshow_images/ folder. Transition effects are configurable:

  • Fade — smooth opacity transition (recommended).
  • Slide — horizontal movement between images.
  • None — instant cut with no animation.

See the Image Manager docs for details on uploading and managing images remotely.

Islamic Events

The display shows upcoming Islamic and Hijri calendar events with a countdown in days. Events are defined in events.json and can be customized by the administrator.

Keyboard Controls

KeyAction
F11Toggle fullscreen mode on/off.
EscExit fullscreen.
⚠️ In fullscreen mode, the Admin and Fullscreen interface buttons are hidden for a cleaner display. Press F11 to reveal the control bar.

Date Display

The current date is shown in two formats:

  • Gregorian date in the selected locale format.
  • Hijri date parsed from the JAKIM API response.

The time format (12-hour AM/PM or 24-hour) is configurable in the Admin Panel.
