Image Manager

Solat Interactive includes an embedded Flask web server that lets administrators upload, preview, and delete slideshow images from any device on the local network — no file transfers or USB drives needed.

ℹ️ The web uploader runs on port 5001 and is accessible when the main application is running.

Accessing the Uploader

To reach the image manager from a phone, tablet, or laptop:

  1. Make sure the host PC is running Solat Interactive.
  2. Connect your device to the same WiFi hotspot as the host PC.
  3. Open the Admin Panel on the display. Note the local IP and port shown (e.g., http://192.168.43.128:5001).
  4. On your device's browser, navigate to that URL.
  5. Log in with the admin password (123 by default).

Uploader Features

⬆️ Upload Images

Upload PNG, JPG, or JPEG files directly from your phone or computer. The images are copied immediately into the slideshow folder.

👁️ Preview Thumbnails

View recent uploads in a thumbnail grid before committing. Quickly identify and remove duplicates.

🗑️ Delete Images

Remove unwanted images directly from the browser. Deletions take effect on the display within the next slideshow cycle.

Supported Image Formats

The slideshow engine accepts the following formats:

  • PNG — recommended for transparency and sharp edges.
  • JPG / JPEG — best for photographs and high-resolution images.

GIF is supported for the Adhan animation overlay only, not for standard slides.

⚠️ Very large images may cause performance issues on lower-powered display PCs. Resize images to match or exceed the display resolution before uploading.

Slideshow Display

Images cycle on the left half of the main display. The appearance is controlled by:

  • Interval: How many seconds each image is displayed (Admin Panel setting).
  • Transition: The animation between images — fade, slide, or none.

Troubleshooting

I cannot access the uploader from my phone

Ensure your phone is connected to the same WiFi hotspot as the host PC. Verify port 5001 is not blocked by Windows Firewall. Check that the host IP shown in the Admin Panel matches your network segment.

Uploaded images do not appear on the display

Images appear on the next slideshow cycle. If they never appear, confirm the files were uploaded to the correct slideshow_images/ folder and that the file format is supported.

The uploader shows "Login Failed"

Double-check the admin password. If it was changed recently, update all stored credentials accordingly.
