FAQ

Frequently asked questions about Solat Interactive. If your question isn't answered here, please open an issue on GitHub.

General

What is Solat Interactive?

Solat Interactive is a desktop application designed for screens at surau and mosques. It automatically displays prayer times from the Malaysian JAKIM API, shows a live countdown, plays Adhan and Iqamah audio at the correct times, and rotates a slideshow of images.

Who is it licensed for?

The application is licensed for Surau Al-Huda IPD BaratDaya. Device binding ensures it only runs on authorized machines.

Can I customize the theme?

Yes. Six built-in themes are available, and you can enable auto-cycle to rotate through them daily. See the Themes page for details.

Installation

How do I install it?

Download the latest SolatInterac.exe from the Releases page, extract it, and run the executable. No installer is required.

Is an internet connection required?

Only on the first run to fetch annual prayer times from the JAKIM e-Solat API. After that, times are cached locally.

What is the default admin password?

The default password is 123. Change it immediately after first launch via the Admin Panel.

Operation

How do I exit fullscreen?

Press F11 to toggle fullscreen, or press Esc to exit fullscreen directly.

The Admin and Fullscreen buttons are missing. Why?

These buttons intentionally hide in fullscreen mode for a clean display. Press F11 to reveal the control bar.

How do I upload images from my phone?

Connect your phone to the same WiFi hotspot as the host PC. The Admin Panel displays the uploader URL (e.g., http://192.168.x.x:5001). Navigate to it, log in, and upload images.

Prayer times are wrong. How do I fix?

Verify the zone setting in Admin Panel matches your JAKIM zone code (e.g., PNG01 for Perak). You can also force a re-fetch of the annual prayer data.

Troubleshooting

No audio plays at prayer time

Check that the system volume is unmuted, the Adhan alert option is enabled in the Admin Panel, and that adhan.mp3 and iqamah.mp3 exist in the audio/ folder.

The image uploader is unreachable

Ensure the host PC's mobile hotspot is active and the device is connected to it. Confirm port 5001 is not blocked by Windows Firewall.

The app says "Invalid Device"

The application is locked to the original Windows hardware UUID. An administrator must use the License section in the Admin Panel to bind a new device if hardware has changed.

Slideshow images are not showing

Confirm that PNG/JPG files are placed in the slideshow_images/ directory. A web uploader is available at http://<IP>:5001 for convenience.

The application crashes on startup

Check solat.log in the data directory for details. Common causes include missing audio files, corrupted configuration JSON, or a broken JAKIM API response.

Developer / Advanced

How do I enable developer mode?

Enter the developer password in the Admin Panel to unlock advanced options: direct API endpoint editing, manual annual prayer time fetching, and other debugging utilities.

Can I contribute?

Yes. Fork the repository, create a feature branch, and submit a pull request. Please follow the existing code style and run any provided tests before submitting.
