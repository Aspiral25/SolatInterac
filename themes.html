Themes & Customization

Solat Interactive supports a powerful theme system that controls the color palette of the entire interface. Administrators can choose from built-in themes or enable automatic daily cycling.

Built-in Themes

Six professionally designed themes are included out of the box:

ThemeMoodBest For
redblue Bold contrast High-visibility displays with strong color separation
blueorange Warm + cool balance General purpose, easy on the eyes
purple Elegant & refined Traditional and dignified environments
coldgreen Calming green tones Quiet, meditative spaces
pastel Soft and light Bright rooms, daytime readability
brown Warm and earthy Wooden interiors, traditional décor

Auto-Cycle

Automatically rotate through themes on a daily basis. This is useful for:

  • Keeping the display visually interesting over long periods.
  • Matching the theme to different prayer times or Islamic months.

Auto-cycle is enabled via the Admin Panel.

Configuration Persistence

The active theme and auto-cycle setting are saved in prayer_times_config.json under the theme and auto_cycle_theme fields. Changes take effect immediately upon saving from the Admin Panel.

Time Format

Beyond themes, the display supports two time formats:

  • 12-hour (AM/PM): Common for everyday users.
  • 24-hour: Standard military/international format.

Set globally via the Admin Panel. The format applies to all timestamps on the display.

Design Notes

Themes are defined as Python dictionaries in config.py with color keys for backgrounds, prayer panels, titles, and borders. Each theme is designed to maintain readibility of prayer times against the slideshow background. The countdown animation color is derived from the current theme's accent color.

💡 Tip: If your surau has specific brand colors, you can add a custom theme by extending the theme dictionary in config.py. See the Developer Mode in the Admin Panel for directions.
