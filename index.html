Display System

A modern, fullscreen desktop application for displaying Muslim prayer times, live countdowns, Adhan audio alerts, and dynamic slideshows — designed for surau and mosque environments.

Quick Start Guide Explore Features View All Docs

⏰ Live Prayer Countdown

Real-time countdown with pulsing animations 3 minutes before each prayer. Highlights current and upcoming prayers automatically.

🔊 Audio Alerts

Automated Adhan and Iqamah playback with configurable per-prayer delays. Multi-stage Iqamah animation with full-screen visuals.

🖼️ Slideshow Manager

Rotating image display with fade and slide transitions. A built-in web uploader lets admins manage media from any phone or laptop.

🎨 Themes & Customization

Six built-in color themes with auto-cycle support. Configurable 12/24-hour time format, volume, and display timings.

⚙️ Admin Panel

Password-protected admin dialog for simulation testing, speaker binding, settings management, and developer debugging tools.

📅 Islamic Calendar

Displays both Gregorian and Hijri dates. Shows upcoming Islamic events with countdown days including Eid and Ramadan.
