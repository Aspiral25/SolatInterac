Getting Started

This guide walks you through installing, configuring, and launching Solat Interactive on a Windows PC.

ℹ️ Note: Solat Interactive is built for Windows and uses PyQt6 for the desktop interface.

System Requirements

Before installing, ensure your machine meets the following requirements:

RequirementDetails
Operating SystemWindows 10 / 11
Memory4 GB RAM minimum (8 GB recommended)
DisplayScreen capable of running the app in fullscreen resolution
AudioSpeakers or audio output device for Adhan / Iqamah
InternetRequired on first run to fetch annual prayer times from JAKIM
NetworkWiFi hotspot for local image uploader access (optional)

Installation

There are two ways to install Solat Interactive: using the pre-built executable or running from source.

Option A — Portable Executable (Recommended)

  1. Download the latest SolatInterac.exe from the GitHub Releases page.
  2. Create a new folder (e.g., C:\Program Files\SolatInteractive).
  3. Copy the executable into that folder.
  4. Double-click SolatInterac.exe to run.
⚠️ Tip: When launching for the first time, the application will prompt for the administrator password. The default password is 123.

Option B — Run from Source

If you want to modify the application or run the latest development version:

  1. Clone the repository: 
    git clone https://github.com/Aspiral25/SolatInterac.git
cd SolatInterac
  2. Create a virtual environment: 
    python -m venv venv
venv\Scripts\activate
  3. Install dependencies: 
    pip install -r requirements.txt
  4. Launch the application: 
    python solah.py

First-Run Behavior

On the very first launch, Solat Interactive will automatically:

  1. Create a USER_DATA_DIR under %APPDATA%\SolatDisplay for persistent storage.
  2. Copy adhan.mp3 and iqamah.mp3 from bundled assets into the audio/ folder.
  3. Create a slideshow_images/ directory for image storage.
  4. Copy events.json (Islamic calendar events) from assets/ if not present.
  5. Fetch annual prayer times from the JAKIM e-Solat API. Requires internet connection
  6. Read the stored Windows hardware UUID to enforce device-specific licensing.
  7. Start the embedded Flask web server on port 5001.

Configuring Mobile Hotspot

The image uploader works best when the host PC shares its mobile hotspot. This allows network devices to reach http://<IP>:5001.

  1. Open Settings → Network & Internet → Mobile hotspot on Windows.
  2. Turn on the hotspot and note the IPv4 address.
  3. Open the Admin Panel (Admin button on the display) and verify the hotspot status.
  4. On your phone, connect to the hotspot and navigate to the displayed URL.
