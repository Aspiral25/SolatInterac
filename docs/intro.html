Introduction

Solat Interactive is a modern, fullscreen desktop application designed to display Muslim prayer times and related Islamic information on dedicated screens — typically installed on PCs or media devices at a surau (Malaysian prayer hall).

🎯 Purpose & Audience

The application targets mosque and surau administrators and worshippers who need an automated, visually appealing display of prayer schedules, live countdowns, and audio/visual alerts (Adhan and Iqamah notifications). It eliminates the need for manual timetable updates by fetching official Malaysian prayer times directly from the JAKIM e-Solat API.

Key Features

📺 Fullscreen Display

Optimized for unattended TV or projector screens. Supports F11 fullscreen and auto-hides controls.

⏰ Live Countdown

Real-time countdown with pulsing animations 3 minutes before each prayer. Highlights current and upcoming prayers.

🔊 Audio Alerts

Automated Adhan at prayer time and Iqamah after configurable delays. Multi-stage Iqamah animation sequence.

🖼️ Slideshow

Rotating images with fade and slide transitions. Web-based remote image uploader for phone/laptop management.

📅 Islamic Calendar

Shows both Gregorian and Hijri dates, plus upcoming Islamic events (Ramadan, Eid, etc.) with countdown days.

🎨 Theme System

Six built-in themes with auto-cycle support. Full color customization via Admin Panel.

Architecture

The application uses a client-server hybrid architecture on a single machine:

┌──────────────────────────────────────────┐
│            Main GUI (Main Thread)         │
│            solah.py / solah_vJ1.py         │
│  PyQt6 QMainWindow                        │
│  Timers: Clock (1s), Data Reload (1h)     │
│  Prayer logic, countdown, QMediaPlayer   │
├────────────────┬─────────────────────────┤
│ Admin Dialog   │  Background Flask Thread │
│ (QDialog)      │  web_server.py           │
│ Settings UI    │  Port 5001               │
│ Simulation     │  Image upload/listing    │
│ Hotspot status │                          │
├────────────────┴─────────────────────────┤
│        Persistent Storage (%APPDATA%)     │
│  prayer_times_config.json, audio/,        │
│  slideshow_images/, events.json           │
└──────────────────────────────────────────┘

Licensing & Device Binding

Solat Interactive uses Windows hardware UUID for per-machine licensing. On first run, the app binds to the current device. This prevents unauthorized use on other PCs without administrative reassignment.
