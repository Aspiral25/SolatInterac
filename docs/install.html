Installation

This page provides detailed installation instructions and an overview of the dependencies and first-run behavior for Solat Interactive.

ℹ️ If you haven't already, review the Getting Started guide for a quicker setup walkthrough.

System Requirements

  • OS: Windows 10 / 11
  • RAM: 4 GB minimum (8 GB recommended)
  • Storage: 500 MB free space
  • Audio: Speaker or audio output for Adhan/Iqamah
  • Internet: Required on first launch to fetch JAKIM prayer times
  • Network: Ad-Hoc WiFi hotspot for local image uploader (optional but recommended)

Portable Executable

The simplest installation path uses the pre-built .exe:

  1. Download the latest release from GitHub Releases.
  2. Extract the archive to a secure folder (e.g., C:\Program Files\SolatInteractive\).
  3. Run SolatInterac.exe.

Build from Source

For development or customization:

git clone https://github.com/Aspiral25/SolatInterac.git
cd SolatInterac

python -m venv venv
.\venv\Scripts\activate

pip install -r requirements.txt

python solah.py
⚠️ Default admin password: 123. Change this immediately after first launch via the Admin Panel.

Packaging with PyInstaller

The build configuration is stored in solah.spec. To produce a fresh executable:

pyinstaller solah.spec

The output executable is named MasjidInterac.exe.

Directory Layout

On first run, the app creates the following structure:

%APPDATA%\SolatDisplay\
├── prayer_times_config.json
├── annual_prayer_times.json
├── events.json
├── solat.log
├── audio\
│   ├── adhan.mp3
│   └── iqamah.mp3
├── slideshow_images\
│   └── (your uploaded images)
└── (optional: adhan.gif, adhan.jpg)
