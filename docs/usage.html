Usage Guide

This guide explains how to use Solat Interactive after installation. The application is designed to run unattended on a display screen, automatically managing prayer times, audio alerts, and visual content.

For Viewers (Worshippers)

The primary users — worshippers at the surau — interact with the application passively by viewing the display:

  • Prayer Times: A clean grid shows all seven daily prayer periods with exact times.
  • Countdown: A live countdown highlights the time until the next prayer, pulsing 3 minutes before.
  • Slideshow: Beautiful images rotate on the left side of the screen.
  • Dates & Events: Gregorian and Hijri dates are displayed alongside upcoming Islamic event countdowns.

No input is required from viewers. The application runs continuously and handles everything automatically.

For Administrators

Surau administrators can access the Admin Panel by pressing the Admin button on the interface or opening the dialog. Default password: 123.

What the Admin Can Do

From the Admin Panel, administrators have full control over:

  • Display Settings: Theme selection (or auto-cycle), 12/24-hour time format, volume control.
  • Slideshow: Transition effect (fade, slide, none) and display interval.
  • Iqamah Delays: Per-prayer delay configuration (seconds between Adhan and Iqamah countdown start).
  • Timing: Pre-Adhan alert lead time, Adhan GIF display duration, Iqamah entry delay, final SOLAT hold time.
  • Adhan Control: Enable/disable automated Adhan playback.
  • License: Bind or unbind the application to a specific Windows PC using hardware UUID.
  • Testing: Simulate any prayer time to test alerts, countdowns, and overlays without waiting for real time.
  • Audio Testing: Play Adhan and Iqamah sounds directly from the panel to verify volume and speaker configuration.

Remote Image Management

The app includes an embedded Flask web server (port 5001) that allows admins to upload and manage slideshow images from any device on the local network:

  1. Connect your phone or laptop to the surau's WiFi hotspot.
  2. Navigate to http://<IP>:5001 (the IP is shown in the Admin Panel).
  3. Enter the password (123 by default).
  4. Upload images, preview thumbnails, and delete unwanted files.

See the Image Manager documentation for detailed instructions.
